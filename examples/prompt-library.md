# Prompt Library

Use these as starting points when exercising Council in different harnesses.

## Architecture

- Codex:
  `Use $council to stress test whether we should split the worker into a separate service before the first pilot.`
- Claude Code:
  `/council Should we keep one shared Postgres schema or split into product-specific schemas before launch?`
- Amp:
  `Use the council skill to deliberate on whether we should split the ingestion service before the first customer.`

## Strategy

- Codex:
  `Use $council to debate whether we should narrow to a single lighthouse customer segment this quarter.`
- Claude Code:
  `/council Should we lead with reliability messaging or cost messaging for the first outbound campaign?`
- Amp:
  `Use the council skill to evaluate whether we should ship the narrow pilot now or wait for broader automation coverage.`

## Debugging

- Codex:
  `Use $council to debate the most likely root cause of this flaky retry path and what to instrument next.`
- Claude Code:
  `/council We have intermittent duplicate jobs after resume. What hypotheses survive scrutiny and what should we test next?`
- Amp:
  `Use the council skill to examine this retry bug and recommend the highest-leverage next debugging step.`
